Introduction

What is UnseenCode?

UnseenCode is a novel way to hide informations in normal pictures.

With the most common desktops and mobile phones,anyone can encode and decode on any pictures at hand.

The code is "Unseen" to human beings while visible to cameras, which makes it easily to be extracted by an APK.

Full Process

  • Select information and pictures to be encoded on the desktop.
  • The program will generate pictures and then display.
  • Set the exposure time in APK according to the refresh rate of the monitor.
  • Press "SCAN" button and get the decoding result.

    • Principle

    How It Works?

    The program first converts the picture into CIE color space and divides the picture into blocks. Then it embeds 0 or 1 in the X component of each block by generating a corresponding frame pair. Finally the frame pair with information are converted back to RGB space.

    While displaying the frame pair alternatively more frequently than Critical Flicker Frequency(CFF), the frame pairs will fuse into the original image, which seems "Unseen" to human beings.

    However, the camera on smart phones ia able to capture the frames independently, as the exposure time is set small enough. The decoder also first divides the frame and converts it into CIE space. Then it calculates △ X, which represents the difference between the X component and Z component of one captured frame. After comparing △ X with the templates of 0 and 1, the decoder determines the bits embedded in the current block, as well as the whole image.
